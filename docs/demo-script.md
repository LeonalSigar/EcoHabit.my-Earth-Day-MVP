# EcoHabit.my Demo Script

## 45-Second Version

Start on the welcome modal and hero:

“This is EcoHabit.my, a Malaysia-flavoured Earth Day action app. I wanted it to feel rooted in everyday habits here, but still completely readable to global judges.”

Dismiss the modal and show the action picker:

“Instead of another generic carbon dashboard, it gives practical actions around commuting, food, reusables, energy, and waste.”

Complete 2 or 3 actions:

“As actions are completed, points and streak update instantly. The local context is built into the action labels and descriptions.”

Scroll to the pledge wall:

“The social layer is the pledge wall. Users can add one anonymous action commitment, and the feed is pre-seeded so the app never feels empty on first load.”

End on the result card:

“The product is designed for a shareable screenshot moment. The point is not perfect measurement, it is making climate-friendly behaviour feel concrete and repeatable.”
